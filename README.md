# COVID-19 Medical Data Analysis

## Overview
This project analyzes real-world COVID-19 medical data to identify infection trends, mortality rates, hospitalizations, and vaccination impact. The project applies data science techniques learned from Cisco NetAcad courses:
- Introduction to Data Science
- Data Analytics Essentials

## Dataset
Source: Our World in Data (Global COVID-19 Dataset) https://ourworldindata.org/covid-cases
- Not including data-set as it is too large

## Objectives
- Analyze COVID-19 case growth
- Study mortality trends
- Explore hospital and ICU demand
- Investigate vaccination impact

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
